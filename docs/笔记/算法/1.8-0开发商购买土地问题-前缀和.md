# KM44.开发商购买土地

## KM44.开发商购买土地

[卡码网题目链接](https://kamacoder.com/problempage.php?pid=1044)

###### 题目描述

在一个城市区域内，被划分成了n * m个连续的区块，每个区块都拥有不同的权值，代表着其土地价值。目前，有两家开发公司，A 公司和 B 公司，希望购买这个城市区域的土地。 

现在，需要将这个城市区域的所有区块分配给 A 公司和 B 公司。

然而，由于城市规划的限制，只允许将区域按横向或纵向划分成两个子区域，而且每个子区域都必须包含一个或多个区块。 为了确保公平竞争，你需要找到一种分配方式，使得 A 公司和 B 公司各自的子区域内的土地总价值之差最小。 

注意：区块不可再分。

###### 输入描述

第一行输入两个正整数，代表 n 和 m。 

接下来的 n 行，每行输出 m 个正整数。

###### 输出描述

请输出一个整数，代表两个子区域内土地总价值之间的最小差距。

###### 输入示例

```
3 3
1 2 3
2 1 3
1 2 3
```

###### 输出示例

```
0
```

###### 提示信息

如果将区域按照如下方式划分：

1 2 | 3
2 1 | 3
1 2 | 3 

两个子区域内土地总价值之间的最小差距可以达到 0。

数据范围：

1 <= n, m <= 100；
n 和 m 不同时为 1。

## 算法思路

如果每次划分都分别重新计算，系统复杂度极高，不能有效利用计算过的信息，我们使用**前缀和存储中间结果**，用空间换时间

分别定义两个数组，一个数组存储前`i`行的所有数和，一个数组存储前`j`列的所有数和

```java
horizontal[i] = horizontal[i - 1] + 新一行所有;
```

```java
vertical[i] = vertical[i - 1] + 新一行所有;
```

从而，只需要遍历一次，后续都可以用中间结果少量计算达成目的

### 实现

```java
import java.util.Scanner;

@SuppressWarnings("ALL")
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int m = sc.nextInt();
        int[][] matrix = new int[n][m];
        int sum = 0; // 所有土地总和
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                matrix[i][j] = sc.nextInt();
                sum += matrix[i][j];
            }
        }
        int []horizontal = new int[n]; // horizontal 水平的
        int []vertical = new int[m]; // vertical 垂直的
        int res = Integer.MAX_VALUE; // 答案
        // 横向
        for (int i = 0; i < n; i++) {
            if (i == 0) {
                horizontal[i] = 0;
            } else {
                horizontal[i] = horizontal[i - 1];
            }
            for (int j = 0; j < m; j++) {
                horizontal[i] += matrix[i][j];
                if (j == m - 1) {
                    res = Math.min(res, Math.abs(sum - horizontal[i] -horizontal[i]));
                }
            }
        }
        // 纵向
        for (int i = 0; i < m; i++) {
            if (i == 0) {
                vertical[i] = 0;
            } else {
                vertical[i] = vertical[i - 1];
            }
            for (int j = 0; j < n; j++) {
                vertical[i] += matrix[j][i];
                if (j == n - 1) {
                    res = Math.min(res, Math.abs(sum - vertical[i] - vertical[i]));
                }
            }
        }
        System.out.println(res);
        sc.close();
    }
}

```
