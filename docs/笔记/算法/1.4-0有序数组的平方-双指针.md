# 977.有序数组的平方

## 977.有序数组的平方-题目描述 

[力扣题目链接(opens new window)](https://leetcode.cn/problems/squares-of-a-sorted-array/)

给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。

示例 1：

- 输入：nums = [-4,-1,0,3,10]
- 输出：[0,1,9,16,100]
- 解释：平方后，数组变为 [16,1,0,9,100]，排序后，数组变为 [0,1,9,16,100]

示例 2：

- 输入：nums = [-7,-3,2,3,11]
- 输出：[4,9,9,49,121]

## 算法思路

1. 正序的话，需要先找正负分界点，分界点需要二分，并且双指针过程中，需要分别处理两个指针越界的情况；具体求解细节见1.3-1双指针
2. 但是如果倒叙，即将两个指针分别放在数组的两端，先找最大的，就会ez很多

## 实际解决

正向也要好好看看，大力出奇迹，包含双指针运动不同情况

```cpp
// 正向
class Solution
{
public:
    vector<int> sortedSquares(vector<int> &nums)
    {
        // 二分找最小大于0
        int left = 0;
        int right = nums.size() - 1;
        int target = nums.size(); // 如果找不到，默认在最后，即全是负数
        while (left <= right)
        {
            int mid = left + ((right - left) >> 1);
            if (nums[mid] >= 0)
            {
                target = mid;
                right = mid - 1;
            }
            else
            {
                left = mid + 1;
            }
        }
        int first = target - 1;
        int second = target;
        vector<int> res;
        while (first >= 0 || second <= nums.size() - 1)
        {
            // 双指针一般会出现很多情况，要考虑清楚
            // 包括T884也是最后两个指针可能会出现一方越界
            if (first < 0)
            {
                res.push_back(nums[second] * nums[second]);
                second++;
                continue;
            }
            else if (second > nums.size() - 1)
            {
                res.push_back(nums[first] * nums[first]);
                first--;
                continue;
            }
            else
            {
                int f = nums[first] * nums[first];
                int s = nums[second] * nums[second];
                if (f < s)
                {
                    res.push_back(f);
                    first--;
                }
                else
                {
                    res.push_back(s);
                    second++;
                }
            }
        }
        return res;
    }
};
```

下面是比较巧妙的逆向

```cpp
// 逆向
class Solution
{
public:
    vector<int> sortedSquares(vector<int> &nums)
    {
        int first = 0;
        int last = nums.size() - 1;
        vector<int> result(nums.size());
        int index = nums.size() - 1;
        while (first <= last)
        {
            int squareFirst = nums[first] * nums[first];
            int squareLast = nums[last] * nums[last];
            if (squareFirst > squareLast)
            {
                result[index--] = squareFirst;
                first++;
            }
            else
            {
                result[index--] = squareLast;
                last--;
            }
        }
        return result;
    }
};
```
