# 707.设计链表

## 707.设计链表

[力扣题目链接](https://leetcode.cn/problems/design-linked-list/description/)

你可以选择使用单链表或者双链表，设计并实现自己的链表。

单链表中的节点应该具备两个属性：`val` 和 `next` 。`val` 是当前节点的值，`next` 是指向下一个节点的指针/引用。

如果是双向链表，则还需要属性 `prev` 以指示链表中的上一个节点。假设链表中的所有节点下标从 **0** 开始。

实现 `MyLinkedList` 类：

- `MyLinkedList()` 初始化 `MyLinkedList` 对象。
- `int get(int index)` 获取链表中下标为 `index` 的节点的值。如果下标无效，则返回 `-1` 。
- `void addAtHead(int val)` 将一个值为 `val` 的节点插入到链表中第一个元素之前。在插入完成后，新节点会成为链表的第一个节点。
- `void addAtTail(int val)` 将一个值为 `val` 的节点追加到链表中作为链表的最后一个元素。
- `void addAtIndex(int index, int val)` 将一个值为 `val` 的节点插入到链表中下标为 `index` 的节点之前。如果 `index` 等于链表的长度，那么该节点会被追加到链表的末尾。如果 `index` 比长度更大，该节点将 **不会插入** 到链表中。
- `void deleteAtIndex(int index)` 如果下标有效，则删除链表中下标为 `index` 的节点。

**示例：**

```java
输入
["MyLinkedList", "addAtHead", "addAtTail", "addAtIndex", "get", "deleteAtIndex", "get"]
[[], [1], [3], [1, 2], [1], [1], [1]]
输出
[null, null, null, null, 2, null, 3]

解释
MyLinkedList myLinkedList = new MyLinkedList();
myLinkedList.addAtHead(1);
myLinkedList.addAtTail(3);
myLinkedList.addAtIndex(1, 2);    // 链表变为 1->2->3
myLinkedList.get(1);              // 返回 2
myLinkedList.deleteAtIndex(1);    // 现在，链表变为 1->3
myLinkedList.get(1);              // 返回 3
```

**提示：**

- `0 <= index, val <= 1000`
- 请不要使用内置的 LinkedList 库。
- 调用 `get`、`addAtHead`、`addAtTail`、`addAtIndex` 和 `deleteAtIndex` 的次数不超过 `2000` 。

## 算法思路

**这里单链表不要忘记`size`变量的更新！！！**

**双链表从链表尾部进行查找时注意下标`index`需要更新为`size-index`或`size-index-1`**

> 注意`addAtIndex(int index, int val) `将一个值为 `val `的节点插入到链表中下标为 index 的节点之前。如果 index 等于链表的长度，那么该节点会被追加到链表的末尾。如果 index 比长度更大，该节点将 不会插入 到链表中。因为在该元素之前，所以`index = size - index`，非`size - index - 1`
>
>             current = tail;
>             index = size - index;
>             while (index > 0) {
>                 current = current.pre;
>                 index--;
>             }

我们这里依旧采取设置**虚拟头节点**的方式进行操作，只不过需要注意的是，在对双向链表进行初始化时，需要虚拟头尾相互指向

```java
head.next = tail;
tail.pre = head;
```

因为插入新节点时如果进行头插或尾插，需要用到**头指尾寻尾**或者**尾指头寻头**，eg头插:

```java
// 自己指别人
newNode.next = head.next;
newNode.pre = head;
// 别人指向它
head.next.pre = newNode; // 这一步会出null.pre的问题
hext.next = newNode;
```

eg尾插：

```java
// 自己指别人
newNode.next = tail;
newNode.pre = tail.pre;
// 别人指它
tail.pre.next = newNode; // 这一步会null.next
tail.pre = newNode;
```

另外在构造`ListNode`设置默认构造方法对val赋值0

这更多是一种**编码习惯和教学意图**，而不是技术上的必须：

1. **显式表达“这个值无关紧要”**
   故意放一个 `0`，是在无声地告诉你：“虚拟节点的值不重要，所以我随便塞了个 0，你千万别用它。”
   如果只写 `new ListNode()`，容易让初学者误以为这个节点**没有值**，但实际上它是有值的，只是默认 0 而已。显式写 `0` 能消除这种误解。

2. **统一构造器，方便以后维护**
   只提供一个 `ListNode(int val)` 构造器，就**强制**创建任何节点时都必须传入一个值。这样能防止不小心创建出一个“半成品”节点（比如忘了设值）。
   在教学环境中，这种“强制”可以帮助学生养成好习惯：**每个节点出生时，身份（val）就必须明确**，哪怕是虚拟节点，也明确给它一个“无用”的 0。

3. **避免依赖默认值带来的潜在混淆**
   虽然 `int` 默认是 0，但如果是 `Integer`（包装类），默认是 `null`，差别就大了。老师可能在潜意识里培养一种风格：**对象的状态要显式初始化**，不依赖语言默认值。这种风格在大型项目中能减少 bug。

   > Integer比较时自动拆箱
   >
   > ```java
   > Integer a = null;
   > int target = 5;
   > System.out.println(a == target);  // 💥 NullPointerException
   > ```
   >
   > 当 `Integer` 和 `int` 用 `==` 比较时，Java 会**自动把 `Integer` 拆箱成 `int`**，调用 `a.intValue()` 方法。
   > 而 `a` 是 `null`，相当于你试图从一个空盒子里倒出豆子，结果手插空了——**空指针异常**。
   >
   > 在链表操作中，很多查找、删除逻辑会写：
   >
   > ```java
   > if (cur.val == targetValue) { ... }
   > ```
   >
   > 这里的 `targetValue` 如果是 `int`（基本类型），`cur.val` 如果是 `null`，就立刻炸。

### 实现

单链表：

```java
class MyLinkedList {
    static class ListNode {
        int val;
        ListNode next;
        ListNode(int val) {
            this.val = val;
        }
    }

    // 虚拟头节点
    private final ListNode head;
    // 记录链表大小
    private int size;

    // 链表初始化
    public MyLinkedList() {
        this.head = new ListNode(0);
        this.size = 0;
    }

    // int get(int index) 获取链表中下标为 index 的节点的值。如果下标无效，则返回 -1 。
    public int get(int index) {
        if (index < 0 || index >= size) {
            return -1;
        }
        ListNode current = head;
        while (index >= 0) {
            current = current.next;
            index--;
        }
        return current.val;
    }

    // void addAtHead(int val) 将一个值为 val 的节点插入到链表中第一个元素之前。在插入完成后，新节点会成为链表的第一个节点。
    public void addAtHead(int val) {
        ListNode newNode = new ListNode(val);
        newNode.next = head.next;
        head.next = newNode;
        size++;
    }

    // void addAtTail(int val) 将一个值为 val 的节点追加到链表中作为链表的最后一个元素。
    public void addAtTail(int val) {
        ListNode newNode = new ListNode(val);
        ListNode current = head;
        while (current.next != null) {
            current = current.next;
        }
        current.next = newNode;
        size++;
    }

    // void addAtIndex(int index, int val) 将一个值为 val 的节点插入到链表中下标为 index 的节点之前。如果 index 等于链表的长度，那么该节点会被追加到链表的末尾。如果 index 比长度更大，该节点将 不会插入 到链表中。
    public void addAtIndex(int index, int val) {
        if (index < 0 || index > size) {
            return;
        }
        ListNode newNode = new ListNode(val);
        ListNode current = head;
        while (index > 0) {
            current = current.next;
            index--;
        }
        newNode.next = current.next;
        current.next = newNode;
        size++;
    }

    // void deleteAtIndex(int index) 如果下标有效，则删除链表中下标为 index 的节点。
    public void deleteAtIndex(int index) {
        if (index < 0 || index >= size) {
            return;
        }
        ListNode current = head;
        while (index > 0) {
            current = current.next;
            index--;
        }
        current.next = current.next.next;
        size--;
    }
}
```

双链表

```java
class MyLinkedList {
    static class ListNode {
        int val;
        ListNode pre,next;
        ListNode(int val) {
            this.val = val;
        }
    }

    // 虚拟头节点
    private final ListNode head;
    // 虚拟尾节点
    private final ListNode tail;
    // 记录链表大小
    private int size;

    // 链表初始化
    public MyLinkedList() {
        this.head = new ListNode(0);
        this.tail = new ListNode(0);
        this.size = 0;
        // 头尾相互指 不然添加第一个节点时报错
        // 不这样就相当于单链表了
        this.head.next = this.tail;
        this.tail.pre = this.head;
    }

    // int get(int index) 获取链表中下标为 index 的节点的值。如果下标无效，则返回 -1 。
    public int get(int index) {
        if (index < 0 || index >= size) {
            return -1;
        }
        // 通过size判断距离哪边更近
        ListNode current = head; // 默认情况下 从头指针开始寻找
        if (index <= size / 2) {
            while (index >= 0) {
                current = current.next;
                index--;
            }
        } else {
           current = tail;
           index = size - index - 1;
           while (index >= 0) {
               current = current.pre;
               index--;
           }
        }
        return current.val;
    }

    // void addAtHead(int val) 将一个值为 val 的节点插入到链表中第一个元素之前。在插入完成后，新节点会成为链表的第一个节点。
    public void addAtHead(int val) {
        ListNode newNode = new ListNode(val);
        newNode.next = head.next;
        head.next.pre = newNode;
        head.next = newNode;
        newNode.pre = head;
        size++;
    }

    // void addAtTail(int val) 将一个值为 val 的节点追加到链表中作为链表的最后一个元素。
    public void addAtTail(int val) {
        ListNode newNode = new ListNode(val);
        newNode.pre = tail.pre;
        tail.pre.next = newNode;
        tail.pre = newNode;
        newNode.next = tail;
        size++;
    }

    // void addAtIndex(int index, int val) 将一个值为 val 的节点插入到链表中下标为 index 的节点之前。如果 index 等于链表的长度，那么该节点会被追加到链表的末尾。如果 index 比长度更大，该节点将 不会插入 到链表中。
    public void addAtIndex(int index, int val) {
        if (index < 0 || index > size) {
            return;
        }
        ListNode newNode = new ListNode(val);
        ListNode current = head;
        if (index <= size / 2) {
            while (index > 0) {
                current = current.next;
                index--;
            }
            newNode.next = current.next;
            current.next.pre = newNode;
            current.next = newNode;
            newNode.pre = current;
        } else {
            current = tail;
            index = size - index;
            while (index > 0) {
                current = current.pre;
                index--;
            }
            newNode.pre = current.pre;
            current.pre.next = newNode;
            newNode.next = current;
            current.pre = newNode;
        }
        size++;
    }

    // void deleteAtIndex(int index) 如果下标有效，则删除链表中下标为 index 的节点。
    public void deleteAtIndex(int index) {
        if (index < 0 || index >= size) {
            return;
        }
        ListNode current = head;
        if (index <= size / 2) {
            while (index > 0) {
                current = current.next;
                index--;
            }
            current.next = current.next.next;
            current.next.pre = current;
        } else {
            current = tail;
            index = size - index - 1;
            while (index > 0) {
                current = current.pre;
                index--;
            }
            current.pre = current.pre.pre;
            current.pre.next = current;
        }
        size--;
    }
}
```
