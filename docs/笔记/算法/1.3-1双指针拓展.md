# 26.删除有序数组中的重复项

## 26.删除有序数组中的重复项-题目描述

[26.删除排序数组中的重复项(opens new window)](https://leetcode.cn/problems/remove-duplicates-from-sorted-array/)

给你一个 **非严格递增排列** 的数组 `nums` ，请你**[ 原地](http://baike.baidu.com/item/原地算法)** 删除重复出现的元素，使每个元素 **只出现一次** ，返回删除后数组的新长度。元素的 **相对顺序** 应该保持 **一致** 。然后返回 `nums` 中唯一元素的个数。

考虑 `nums` 的唯一元素的数量为 `k`。去重后，返回唯一元素的数量 `k`。

`nums` 的前 `k` 个元素应包含 **排序后** 的唯一数字。下标 `k - 1` 之后的剩余元素可以忽略。

**判题标准:**

系统会用下面的代码来测试你的题解:

```
int[] nums = [...]; // 输入数组
int[] expectedNums = [...]; // 长度正确的期望答案

int k = removeDuplicates(nums); // 调用

assert k == expectedNums.length;
for (int i = 0; i < k; i++) {
    assert nums[i] == expectedNums[i];
}
```

如果所有断言都通过，那么您的题解将被 **通过**。

**示例 1：**

```
输入：nums = [1,1,2]
输出：2, nums = [1,2,_]
解释：函数应该返回新的长度 2 ，并且原数组 nums 的前两个元素被修改为 1, 2 。不需要考虑数组中超出新长度后面的元素。
```

**示例 2：**

```
输入：nums = [0,0,1,1,1,2,2,3,3,4]
输出：5, nums = [0,1,2,3,4,_,_,_,_,_]
解释：函数应该返回新的长度 5 ， 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4 。不需要考虑数组中超出新长度后面的元素。
```

**提示：**

- `1 <= nums.length <= 3 * 104`
- `-100 <= nums[i] <= 100`
- `nums` 已按 **非递减** 顺序排列。

## 算法思路

依旧一个慢指针维护确定元素，快指针向前快速搜索，遍历快指针，找到~~重复元素（num[fast]==num[fast-1]）~~ 非重复元素（num[fast] != num[slow]）即可更新慢指针

## 实际求解

```cpp
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int slowIdx = 0;
        for (int fastIdx = 1; fastIdx < nums.size(); ++fastIdx) {
            if (nums[fastIdx] != nums[slowIdx]) {
                ++slowIdx; // 注意代码细节，先移动慢指针，再赋值
                nums[slowIdx] = nums[fastIdx];
            }
        }
        return slowIdx + 1;
    }
};
```

```java
                nums[++slowIndex] = nums[fastIndex];
            }
        }
        return slowIndex + 1;
    }
```

# 283.移动零

## 283.移动零-题目描述

给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**请注意** ，必须在不复制数组的情况下原地对数组进行操作。

**示例 1:**

```
输入: nums = [0,1,0,3,12]
输出: [1,3,12,0,0]
```

**示例 2:**

```
输入: nums = [0]
输出: [0]
```

**提示**:

- `1 <= nums.length <= 104`
- `-231 <= nums[i] <= 231 - 1`

**进阶：**你能尽量减少完成的操作次数吗

## 算法思路

移动元素那题的`target`为0版本，依旧双指针;

唯一多点的操作是把`slowIdx`后的元素也赋值为0

## 实际求解

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slowIdx = 0;
        for (int fastIdx = 0; fastIdx < nums.size(); ++fastIdx) {
            if (nums[fastIdx] != 0) {
                nums[slowIdx] = nums[fastIdx];
                ++slowIdx;
            }
        }
        for (int i = slowIdx; i < nums.size(); ++i) {
            nums[i] = 0;
        }
    }
};
```

```java
class Solution {

            if (nums[fast] != 0) {
                nums[slow++] = nums[fast];
            }
        }
        while (slow < nums.length) {
            nums[slow++] = 0;
        }
    }
}
```

# 844.比较含退格的字符串

## 844.比较含退格的字符串-题目描述

[844.比较含退格的字符串(opens new window)](https://leetcode.cn/problems/backspace-string-compare/)

给定 `s` 和 `t` 两个字符串，当它们分别被输入到空白的文本编辑器后，如果两者相等，返回 `true` 。`#` 代表退格字符。

**注意：**如果对空文本输入退格字符，文本继续为空。

**示例 1：**

```
输入：s = "ab#c", t = "ad#c"
输出：true
解释：s 和 t 都会变成 "ac"。
```

**示例 2：**

```
输入：s = "ab##", t = "c#d#"
输出：true
解释：s 和 t 都会变成 ""。
```

**示例 3：**

```
输入：s = "a#c", t = "b"
输出：false
解释：s 会变成 "c"，但 t 仍然是 "b"。
```

**提示：**

- `1 <= s.length, t.length <= 200`
- `s` 和 `t` 只含有小写字母以及字符 `'#'`

**进阶：**

- 你可以用 `O(n)` 的时间复杂度和 `O(1)` 的空间复杂度解决该问题吗？

## 算法思路

双指针，快指针搜索，慢指针模拟

> 还有就是返回结果时使用方法`substr(0,slowPos)`截取
>
> 实际写下来其实慢指针屌用没有·不如直接用栈模拟

实际上有更优秀的方法：

1. 使用栈进行模拟，在遇到`#`,且`str`非空时将栈顶元素`pop_back`出来
2. 如果没遇到`#`，正常`push_back`进去

要想双指针，首先要考虑两个指针不能走回头路，想让两个指针不走回头路，可以用倒序遍历的方法实现，其中两个指针分别代表逆序时两个字符串确定的位置。

因为字母是否被删除只受到后面的`#`的影响，不受前面`#`的影响，我们遇到`#`记录下来即可知道要跳过多少字母，这样可以实现指针不用走回头路。

具体寻找双指针的实现代码可以关注下，小模拟

## 实际求解

```cpp
// 最蠢的双指针模拟
class Solution {
public:
    bool backspaceCompare(string s, string t) {
        return build(s) == build(t);
    }
private:
    string build(const string& str) {
        string result;
        int slowIdx = 0;
        for (int fastIdx = 0; fastIdx < str.size(); ++fastIdx) {
            if (str[fastIdx] != '#') {
                result.push_back(str[fastIdx]);
                ++slowIdx;
            } else {
                if (slowIdx > 0) {
                    --slowIdx;
                    result.pop_back();
                }
            }
        }
        // return result.substr(0, slowIdx);
        return result;
    }
};
```

```cpp
// 逆序双指针
class Solution {
public:
    bool backspaceCompare(string s, string t) {
        int i = s.size() - 1, j = t.size() - 1;
        int skipS = 0, skipT = 0;

        while (i >= 0 || j >= 0) {
            // 找到有效i
            while (i >= 0) {
                if (s[i] == '#') {
                    skipS++;
                    i--;
                } else if (skipS > 0) {
                    skipS--;
                    i--;
                } else { // 找到了有效元素
                    break;
                }
            }
			// 找到有效j
            while (j >= 0) {
                if (t[j] == '#') {
                    skipT++;
                    j--;
                } else if (skipT > 0) {
                    skipT--;
                    j--;
                } else {
                    break;
                }
            }

            if (i >= 0 && j >= 0) {
                // 正常比较
                if (s[i] != t[j]) {
                    return false;
                }
            } else {
                // 有一个字符串有效区遍历完了，另一个没遍历完
                // 说明两个字符串有效长度不同
                if (i >= 0 || j >= 0) {
                    return false;
                }
            }
            i--;
            j--;
        }
        return true;
    }
};
```

# 977.有序数组的平方

## 977.有序数组的平方-题目描述

[977.有序数组的平方(opens new window)](https://leetcode.cn/problems/squares-of-a-sorted-array/)

给你一个按 **非递减顺序** 排序的整数数组 `nums`，返回 **每个数字的平方** 组成的新数组，要求也按 **非递减顺序** 排序。

**示例 1：**

```
输入：nums = [-4,-1,0,3,10]
输出：[0,1,9,16,100]
解释：平方后，数组变为 [16,1,0,9,100]
排序后，数组变为 [0,1,9,16,100]
```

**示例 2：**

```
输入：nums = [-7,-3,2,3,11]
输出：[4,9,9,49,121]
```

**提示：**

- `1 <= nums.length <= 104`
- `-104 <= nums[i] <= 104`
- `nums` 已按 **非递减顺序** 排序

## 算法思路

先二分法找正负交界（最小的大于零的数），从该位置起，维护两个指针，仅当其中一个被放入`vector`时，移动该指针

- 实际写代码时，在二分找`target`时，应该给`target`赋初值为找不到`target`时的假定位置，此处为`nums.size()`,而非0
- 还有就是双指针移动时，经常会出现一个指针遍历完越界，另一个指针还在走的情况，注意全面考虑所有情况，这题和T844题均出现了一个指针越界的情况，特别注意！！！
- 另有奇思表示左右指针可以放在数组的两端，这样甚至省去了二分找分界点的时间！！！具体代码用java放附录里了，cpp即使不这样也是超过100%

## 实际求解

```cpp
class Solution
{
public:
    vector<int> sortedSquares(vector<int> &nums)
    {
        // 二分找最小大于0
        int left = 0;
        int right = nums.size() - 1;
        int target = nums.size(); // 如果找不到，默认在最后，即全是负数
        while (left <= right)
        {
            int mid = left + ((right - left) >> 1);
            if (nums[mid] >= 0)
            {
                target = mid;
                right = mid - 1;
            }
            else
            {
                left = mid + 1;
            }
        }
        int first = target - 1;
        int second = target;
        vector<int> res;
        while (first >= 0 || second <= nums.size() - 1)
        {
            // 双指针一般会出现很多情况，要考虑清楚
            // 包括T884也是最后两个指针可能会出现一方越界
            if (first < 0)
            {
                res.push_back(nums[second] * nums[second]);
                second++;
                continue;
            }
            else if (second > nums.size() - 1)
            {
                res.push_back(nums[first] * nums[first]);
                first--;
                continue;
            }
            else
            {
                int f = nums[first] * nums[first];
                int s = nums[second] * nums[second];
                if (f < s)
                {
                    res.push_back(f);
                    first--;
                }
                else
                {
                    res.push_back(s);
                    second++;
                }
            }
        }
        return res;
    }
};
```

```java
class Solution {
    public int[] sortedSquares(int[] nums) {
        int left = 0, right = nums.length - 1;
        int target = nums.length;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] >= 0) {
                target = mid;
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        int first = target - 1;
        int second = target;
        int []res = new int[nums.length];
        int index = 0;
        while (first >= 0 || second < nums.length) {
            if (first < 0) {
                res[index++] = nums[second]*nums[second];
                second++;
            } else if (second >= nums.length) {
                res[index++] = nums[first]*nums[first];
                first--;
            } else {
                if (nums[first] * nums[first] > nums[second] * nums[second]) {
                   res[index++] = nums[second]*nums[second];
                   second++;
                } else {
                    res[index++] = nums[first]*nums[first];
                    first--;
                }
            }
        }
        return res;
    }
}
```

```java
// 左右指针分别指向数组的两端，省去了二分的过程！！！！
class Solution {
    public int[] sortedSquares(int[] nums) {
        // 双指针，数组从小到大排序，数组中间的平方后肯定是最小的，两头的是最大的
        // 定义一个新数组，和原数组长度一样，比较原数组两头谁最大，把最大的放在新数组的最后一位，
        //原数组谁最大就往前或往后移动一位
        int l = 0;
        int r = nums.length - 1;
        int [] result = new int [nums.length];
        int index = result.length - 1;
        while (l <= r){
            if(nums[l] * nums[l] > nums[r] * nums[r]){
               result[index] = nums[l] * nums[l];
               index --;
               l ++;
            }else {
                 result[index] = nums[r] * nums[r];
                 index --;
                 r --;
            }
        }
        return result;
    }
}
```
