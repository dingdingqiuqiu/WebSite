# 76.最小覆盖子串

## 76.最小覆盖字串

[力扣题目链接](https://leetcode.cn/problems/minimum-window-substring/)

给定两个字符串 `s` 和 `t`，长度分别是 `m` 和 `n`，返回 s 中的 **最短窗口 子串**，使得该子串包含 `t` 中的每一个字符（**包括重复字符**）。如果没有这样的子串，返回空字符串 `""`。

测试用例保证答案唯一。 

**示例 1：**

```
输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"
解释：最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。
```

**示例 2：**

```
输入：s = "a", t = "a"
输出："a"
解释：整个字符串 s 是最小覆盖子串。
```

**示例 3:**

```
输入: s = "a", t = "aa"
输出: ""
解释: t 中两个字符 'a' 均应包含在 s 的子串中，
因此没有符合条件的子字符串，返回空字符串。
```

**提示：**

- `m == s.length`
- `n == t.length`
- `1 <= m, n <= 105`
- `s` 和 `t` 由英文字母组成

**进阶：**你能设计一个在 `O(m + n)` 时间内解决此问题的算法吗？

## 算法思路

> 滑动窗口本质枚举了每个 `right` 作为区间右端点时，满足条件的最短区间。
>
> 因为前一个`left`和`right`已知最短区间，满足大于`target`的条件
>
> 所以后一个`right`（正数）加进来后，前一个`left`和后一个`target`一定也满足大于`target`的条件
>
> 因此只需要尝试`left`右移即可
>
> 以上即为滑动窗口（双指针）为什么有效的原因

本题核心思想依旧是右指针加入新的一定满足`target`，实际上在右指针不动的基础上向右移动左指针直到满足“最小”

至于对结果是否满足的判断过程，可以借助哈希表的数据结构。

## 实际解决

```java
class Solution {
    public String minWindow(String s, String t) {
        if (s == null || t == null || s.length() == 0 || t.length() == 0 || s.length() < t.length()) {
            return "";
        }
        
        // 记录t中每个字符的出现次数
        int[] need = new int[128];
        for (int i = 0; i < t.length(); i++) {
            need[t.charAt(i)]++;
        }
        
        // 滑动窗口边界
        int left = 0, right = 0;
        // 需要匹配的字符种类数
        int needCount = t.length();
        // 记录最小窗口的起始位置和长度
        int minStart = 0, minLen = Integer.MAX_VALUE;
        
        while (right < s.length()) {
            char rightChar = s.charAt(right);
            // 如果当前字符是t中需要的字符
            if (need[rightChar] > 0) {
                needCount--;
            }
            // 将当前字符加入窗口（need数组中对应位置减1）
            need[rightChar]--;
            
            // 当窗口包含t的所有字符时，尝试收缩左边界
            while (needCount == 0) {
                // 更新最小窗口
                if (right - left + 1 < minLen) {
                    minLen = right - left + 1;
                    minStart = left;
                }
                
                char leftChar = s.charAt(left);
                // 如果左边界字符是t中需要的字符
                if (need[leftChar] == 0) {
                    needCount++;
                }
                // 移出左边界字符
                need[leftChar]++;
                left++;
            }
            right++;
        }
        
        return minLen == Integer.MAX_VALUE ? "" : s.substring(minStart, minStart + minLen);
    }
}
```
