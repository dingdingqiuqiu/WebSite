# KM58.区间和

## KM58.区间和

[卡码网题目链接](https://kamacoder.com/problempage.php?pid=1070)

###### 题目描述

给定一个整数数组 Array，请计算该数组在每个指定区间内元素的总和。

###### 输入描述

第一行输入为整数数组 Array 的长度 n，接下来 n 行，每行一个整数，表示数组的元素。随后的输入为需要计算总和的区间下标：a，b （b > = a），直至文件结束。

###### 输出描述

输出每个指定区间内元素的总和。

###### 输入示例

```
5
1
2
3
4
5
0 1
1 3
```

###### 输出示例

```
3
9
```

###### 提示信息

数据范围：
0 < n <= 100000

## 算法思路

注意到查询是多次的，如果每次都要从`arr[a]`加到`arr[b]`复杂度将极高

我们可以**引入前缀和数组`pre[n]`**,其中第`i`项表示从`arr[0]`加到`arr[i]`的总和

由于

```java
pre[i] = pre[i-1] + arr[i] // 方便起见 我们这里使用sum += arr[i]实现 本质一致
```

因此我们只需要遍历一遍`arr`数组即可获得完整前缀和辅助数组`pre`

后续任何从`arr[a]`加到`arr[b]`的操作均可使用一次`pre[b] - pre[a] + arr[a]`实现

从而大大降低了系统复杂度

### 实现

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] arr = new int[n];
        int[] pre = new int[n]; // 前缀和辅助数组 第i个数字是第0到i的和
        int sum = 0;
        for (int i = 0; i < n; i++) {
            arr[i] = sc.nextInt();
            sum += arr[i];
            pre[i] = sum;
        }
        while (sc.hasNext()) {
           int a = sc.nextInt();
           int b = sc.nextInt();
           int res = pre[b] - pre[a] + arr[a];
           System.out.println(res);
        }
        sc.close();
    }
}
```
