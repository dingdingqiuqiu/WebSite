# 35.搜索插入位置

## 35.搜索插入位置-问题描述

[力扣题目链接(opens new window)](https://leetcode.cn/problems/search-insert-position/) 35

给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

你可以假设数组中无重复元素。

示例 1:

- 输入: [1,3,5,6], 5
- 输出: 2

示例 2:

- 输入: [1,3,5,6], 2
- 输出: 1

示例 3:

- 输入: [1,3,5,6], 7
- 输出: 4

示例 4:

- 输入: [1,3,5,6], 0
- 输出: 0

## 算法思路

二分查找（704）那道题找不到时~~返回mid即可~~ 返回`left` 或`right+1`

原因解释：

假设进行n次处理后循环结束，没有找到`target`，此时`left`和`right`的相对位置一定是

```
left == right + 1
```

然而`mid`此时的位置却是根据第**n-1**次`left`和`right`的值计算得到的（此时`mid` == `left(n-1次)` == `right（n-1次）`，三者相等）

通过第n次比较，才知道`target`和第n-1次`mid(left,right)`的大小关系，从而确定更新`right`还是`left`

eg.若`target`在所有元素前

第n-1次：`mid`=`left` = `right` =0

第n次：`right`=-1 `left` = 0 `mid` =0

eg.若`target`在所有元素后,s为数组长度

第n-1次：`mid`=`left` = `right` = s-1

第n次：`right`=s-1 `left` = s `mid` =s-1

同理，若`target`在中间位置，要比较的元素位置为x

第n-1次：`mid`=`left` = `right` = x

第n次分两种情况：

1. x > target:`right`=x-1 `left` = x `mid` =x,此时`target`要插入的位置即为x
2. x < target:`right`=x `left` = x+1 `mid` =x,此时`target`要插入的位置即为x+1

## 实际求解

```cpp
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int left = 0; int right = nums.size() - 1;
        while (left <= right){
            int mid = left + ((right - left) >> 1);
            if (nums[mid] > target) {
                right = mid - 1;
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                return mid;
            }
            /* code */
        }
        return left; 
    }
};
```

# 34. 在排序数组中查找元素的第一个和最后一个位置

## 34. 在排序数组中查找元素的第一个和最后一个位置-问题描述

[力扣链接(opens new window)](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/)

给定一个按照升序排列的整数数组 nums，和一个目标值 target。找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 target，返回 [-1, -1]。

进阶：你可以设计并实现时间复杂度为 $O(\log n)$ 的算法解决此问题吗？

示例 1：

- 输入：nums = [5,7,7,8,8,10], target = 8
- 输出：[3,4]

示例 2：

- 输入：nums = [5,7,7,8,8,10], target = 6
- 输出：[-1,-1]

示例 3：

- 输入：nums = [], target = 0
- 输出：[-1,-1]

## 解题思路

二分法找开始位置；同理二分法找结束位置，时间复杂度为 $O(\log n)$ 

> 如果二分法找开始位置后，遍历，时间复杂度为 $O(\log n + n )$ ，为$O(n)$

我们以找第一个位置的二分法为例子，探究具体代码细节

我们在[left,right]范围内找到`target`时，先假定其为我们要寻找的第一个位置，然后更新`right`为`mid-1`（也就是继续在左半边寻找剩余的`target`）。在新的范围内继续尝试寻找`target`,若能找到，则继续更新。

也就是找到目标元素时，不停下来，继续在左边范围继续寻找，直到找到左边的最后一个`target`为止。结束条件就是左边一直找不到

同理，找后一个位置时，我们找到`target`后，更新`lastPos`的值，然后在右半边寻找剩余的`target`直到找不到为止。

## 实际求解

```cpp
class Solution {
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        int firstPos = findFirstPosition(nums, target);
        int lastPos = findLastPosition(nums, target);
        return {firstPos, lastPos};
    }
private:
    int findFirstPosition(const vector<int>& nums, int target) {
        int left = 0;
        int right = nums.size() - 1;
        int firstPos = -1;

        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == target ) {
                firstPos = mid;
                right = mid - 1; // Continue searching in the left half
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return firstPos;
    }
    int findLastPosition(const vector<int>& nums, int target) {
        int left = 0;
        int right = nums.size() - 1;
        int lastPos = -1;

        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == target) {
                lastPos = mid;
                left = mid + 1; // Continue searching in the right half
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return lastPos;
    }
};
```

    private int searchLast(int[] nums, int target) {

                lastPos = mid;
                left = mid + 1;
            } else if (nums[mid] > target) {
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        return lastPos;
    }
}

# 69.x的平方根

## 69.x的平方根-问题描述

[69.x 的平方根(opens new window)](https://leetcode.cn/problems/sqrtx/)

给你一个非负整数 `x` ，计算并返回 `x` 的 **算术平方根** 。

由于返回类型是整数，结果只保留 **整数部分** ，小数部分将被 **舍去 。**

**注意：**不允许使用任何内置指数函数和算符，例如 `pow(x, 0.5)` 或者 `x ** 0.5` 。

**示例 1：**

```
输入：x = 4
输出：2
```

**示例 2：**

```
输入：x = 8
输出：2
解释：8 的算术平方根是 2.82842..., 由于返回类型是整数，小数部分将被舍去。
```

**提示：**

- `0 <= x <= 231 - 1`

## 解题思路

搜索范围为[1,x]，停止条件为最大的ans使得$ans * ans <= x$

依旧满足条件，记录答案，继续往右半部分搜索是否有更大的ans

## 实际求解

```cpp
class Solution {
public:
    int mySqrt(int x) {
        int left = 0, right = x;
        int ans = -1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            long long square = static_cast<long long>(mid) * mid;
            // 等效为以下，static_cast<long long>(mid)转换了mid的数据类型
            // long long mid = left + (right - left) / 2;
            // long long square = mid * mid;
            if (square <= x) {
                ans = mid; // Update answer
                left = mid + 1; // Try to find a larger mid
            } else {
                right = mid - 1; // Try to find a smaller mid
            }
        }
        return ans;
    }
};
```

# 367.有效的完全平方数

## 367.有效的完全平方数-题目描述

[367.有效的完全平方数(opens new window)](https://leetcode.cn/problems/valid-perfect-square/)

给你一个正整数 `num` 。如果 `num` 是一个完全平方数，则返回 `true` ，否则返回 `false` 。

**完全平方数** 是一个可以写成某个整数的平方的整数。换句话说，它可以写成某个整数和自身的乘积。

不能使用任何内置的库函数，如 `sqrt` 。

**示例 1：**

```
输入：num = 16
输出：true
解释：返回 true ，因为 4 * 4 = 16 且 4 是一个整数。
```

**示例 2：**

```
输入：num = 14
输出：false
解释：返回 false ，因为 3.742 * 3.742 = 14 但 3.742 不是一个整数。
```

**提示：**

- `1 <= num <= 231 - 1`

## 解题思路

搜索范围为[1,num],如果得到的$mid * mid = num$,返回Ture,否则，返回False

## 具体实现

```cpp
class Solution {
public:
    bool isPerfectSquare(int num) {
        int left = 0, right = num;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            long long square = (long long)mid * mid;
            if (square == num) {
                return true;
            } else if (square < num) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return false;
    }
};
```
